Fedora

Normal updates

sudo dnf check-update
sudo dnf upgrade --best --allowerasing -y

Distro Upgrade

Check your current version lsb_release -a

When fedora 36 comes out:

sudo dnf upgrade --refresh
sudo dnf install dnf-plugin-system-upgrade
sudo dnf system-upgrade download --releasever=36 --best --allowerasing
sudo dnf system-upgrade reboot

When 37 comes out, change to releasever=37 etc etc
You can only upgrade 2 at a time. So if you are on 34, you have to upgrade to 36 before 37.
